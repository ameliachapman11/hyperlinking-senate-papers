# Guide to Files Within the Internship Folder
**Overview:** the `documentation` folder consists of documentation regarding the project. As opposed to the `internship` folder, it is more intended to be of usage to the general public, such as those who are looking to learn more on how to utilize the code created or build on the web application. 

</br>

The following is an outline of each document within this folder:
* The [Current Progress](current-progress.md) file outlines which steps of the web application creation and hyperlinking pipeline have currently been completed
* The [Future Steps](future-steps.md) file outlines the next steps to directly be taken in the implementation of the hyperlinking pipeline and development of the web application
* The [Problematic Files](problematic-files.md) file outlines some specific issues encountered when converting from PDF to XML, such as specific instances of tables which cannot be recreated. These issues may impact the way hyperlinking process is approached as clean XML is required for querying and identifying hyperlink targets.
* The [WordPress Guide](wordpress-guide.md) file provides instructions for running the current WordPress site on a LocalWP dev environment and downloading the database of Senate papers
* The [XML Schema](xml-schema.md) file provides a detailed description of the custom XML schema created for the purposes of this project, which has been used for developing the `extract_xml.py` script
